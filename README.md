
. Carditects Studio

Project Idea

Carditects Studio is a creative studio specialized in designing and developing card games. It combines strategic thinking with artistic and interactive experience.

The goal is to build a strong brand capable of competing in both the local and global markets, and to present the first innovative Egyptian card game with a unique and distinctive identity.
